# Screenshots Directory

Place your module screenshots here with these exact filenames:

## Required Screenshots

1. **`screenshot-main-panel.png`**
   - Main panel view showing all combat phases
   - Should show: RUNDA header, all 4 phases (Ranged, Melee, Mystic, Inactive), combatants in different phases
   - Recommended size: ~700x600px (tall format)

2. **`screenshot-engagement.png`**
   - Melee engagement system
   - Should show: Sword icon (attacker), shield icon (defender), engagement buttons (link, toggle, unlink)
   - Recommended size: ~400x500px

3. **`screenshot-engage-dialog.png`**
   - Engagement dialog
   - Should show: Dialog with radio buttons for selecting engagement target
   - Recommended size: ~500x400px

## How to Take Screenshots

1. Open Foundry VTT with your module loaded
2. Set up a combat encounter with multiple combatants in different phases
3. Use Windows Snipping Tool (`Win + Shift + S`) or any screenshot tool
4. Save images as PNG files with the exact filenames listed above
5. Place all files in this directory (`docs/images/`)

The README will automatically display these images once they're saved here.
